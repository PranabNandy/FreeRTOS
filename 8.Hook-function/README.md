## Hook function


In FreeRTOS, an **idle task hook functio**n is a user-defined function that the idle task will call periodically. The idle task is a special task that the FreeRTOS kernel runs when no other tasks are ready to run. The idle task hook allows you to add custom code that executes whenever the idle task is running.
